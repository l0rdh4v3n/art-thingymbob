# Day Schedule Generator

Lightweight planning app that turns a list of desired tasks into a day schedule with lunch and dinner blocks.

## Features

- Add desired tasks with estimated durations
- Edit or delete tasks before generating a plan
- Set day start and end times plus lunch and dinner anchors
- Generate a schedule for the day with open time included
- Save recent generated plans in browser localStorage

## Run

```bash
npm install
npm run dev
```
